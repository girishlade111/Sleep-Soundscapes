# Sleep-Soundscapes
Sleep Soundscapes
**Sleep Soundscapes**  
**Description**: Play ambient noises (rain, forest) for better sleep.  
**Features**:  
- Timer to auto-stop playback.  
- Mix multiple sounds.  
- Offline mode.  
**Tech**: ExoPlayer (Android) / AVFoundation (iOS).  
**No Auth Needed**: No cloud sync.
